Studio Pro Second Semestre 2022 : Correction / Optimisation

5 séances pour proposer après exposé du client formel une charte graphique, un wireframe, des maquettes XD et procéder à l'intégration.
Le choix de la maquette finale a été voté par 2 tours à l'unanimité. Je n'avais pas voté pour celle-ci. J'ai du optimiser le code travaillé en équipe pour qu'il soit maintenable, simple et optimisé pour le RWD

L'enjeu consistait à optimiser le code de manière à pouvoir switcher d'une mise en page avec du display:inline-block à une mise en page avec Flexbox, en quelques lignes, grâce au regroupement de sélecteurs utiles. Un travail indispensable selon moi. Puisqu'il oblige à traquer les répétitions dans le code et optimise les pages pour gérer facilement le responsive.
